# Publish the menu website

This is a static website. The files that must be uploaded are:

- `index.html`
- `styles.css`
- `script.js`
- `assets/menu-hero.png`
- `.nojekyll`

## Fastest option: Netlify Drop

1. Go to https://app.netlify.com/drop
2. Drag the whole `E:\codex_ELY` folder into the page.
3. Netlify will create a public URL.

## GitHub Pages option

1. Create a new GitHub repository.
2. Upload all files in `E:\codex_ELY`.
3. Open the repository settings.
4. Go to Pages.
5. Choose Deploy from branch, then select the main branch and root folder.
6. GitHub will create a public URL.
